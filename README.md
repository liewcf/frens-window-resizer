## Install (unpacked)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the extension folder

## Usage

- Click the extension icon to toggle the current window to the last-selected preset (default: `390x844`).
- Click again to restore the previous window bounds.
- Right-click the extension icon to choose a preset:
  - **Mobile** (`390x844`)
  - **Tablet** (`768x1024`)

## Uninstall

1. Open `chrome://extensions`
2. Find **Frens Window Resizer**
3. Click **Remove** (or toggle it off to disable)
